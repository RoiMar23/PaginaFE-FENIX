# Fenix Frente Estudiantil

Sitio web estático creado con HTML, CSS y JavaScript puro.

## Estructura

- `index.html`: contenido principal de la página.
- `styles.css`: estilos visuales y responsive.
- `script.js`: lógica simple del formulario.

## Cómo verlo en tu PC

1. Abre la carpeta del proyecto en VS Code.
2. Abre `index.html` en el navegador o usa Live Server si lo tienes instalado.
3. Reemplaza los textos, redes y enlaces de WhatsApp por tus datos reales.

## Qué personalizar después

- Link del grupo de WhatsApp.
- Usuario de Instagram, TikTok y Facebook.
- Foto o logo oficial de Fenix si quieres usar una imagen real.
- Texto final de propuestas.

## Publicarlo en GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube estos archivos al repositorio.
3. Activa GitHub Pages en la rama principal.
4. Usa la URL que GitHub te entregue como dominio público.
